● 一言
OggVorbis形式のサウンドを再生するためのライブラリです。
詳しくは同梱のヘルプをご覧ください。

● バージョン履歴
2002/06/21 Ver 0.50  一般公開
2003/06/25 Ver 0.51  Cのラッパー関数群を導入、Ogg Vorbis Win32 SDK のバージョンを1.0に変更
2003/08/09 Ver 0.51a Ogg Vorbis のライセンス条項を追記
2003/09/08 Ver 0.60  関数追加(シーク、全再生時間取得、再生位置取得、再生状態取得、コメント取得)、Cラッパー関数の修正
2004/10/14 Ver 0.61  関数追加(フィールド名指定コメント取得)、まれにフリーズするバグの修正。関数の呼び出し規約をstdcallに統一。C++版とC版のReleaseとDeleteの使い方を一致させた。
